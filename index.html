Code Explanation

The website is built using three technologies:

  • HTML creates the structure of the webpage.
  • CSS styles the page with colors, spacing, and layout.
  • JavaScript adds interactivity like button clicks and form validation.

The showMessage() function displays a welcome message when the button is clicked. The submitForm() function checks if the user entered their name and message. If both fields are filled, it displays a success message.
